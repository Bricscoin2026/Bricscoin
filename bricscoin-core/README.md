# BricsCoin Core

Full node wallet that connects to the real BricsCoin network.

## Install

```bash
npm install
npm start
```

## Features

- Connects to bricscoin26.org (real network)
- Create wallets with seed phrases
- Send real transactions
- Mine real blocks (visible on website)
- View network stats and blocks

## Requirements

- Node.js 18+
- npm
